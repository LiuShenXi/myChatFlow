# ChatFlow 项目总览

## 项目定位

ChatFlow 是一个基于 Next.js 15 的单体全栈 AI 对话应用，重点验证多模型接入、BYOK、认证、会话管理和流式聊天这条主链路。

## 核心使用场景

- 用户登录后创建或自动获得首条会话
- 在设置页保存不同 provider 的 API Key
- 在模型选择器中切换模型并发起真实聊天
- 使用自定义模型配置扩展模型 ID 和 Base URL

## 当前能力范围

- 登录、认证和会话状态已经打通，未登录会有明确的前端提示
- 会话抽屉可以加载、创建、切换和管理当前用户的会话
- 聊天主链路已经可用，消息可以通过流式方式返回并落库
- 设置页可以保存 provider 的 API Key，并支持按 provider 切换模型能力
- 已支持内置模型、OpenAI-compatible provider 以及自定义模型配置
- 基础图片输入链路已经存在，当前可以把图片随消息一起发送
- 豆包这类特殊 provider 已有单独处理逻辑，能通过用户配置的 `endpointId` 完成请求
