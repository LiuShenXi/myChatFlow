# ChatFlow 本地浏览器联调清单

## 使用方式

- 先完成 `npm run doctor`
- 再完成 `npm run verify:local`
- 启动开发服务器：`npm run dev`
- 浏览器联调时，建议同时打开终端输出和浏览器 Network 面板

## 最低通过标准

以下项目通过后，才算本地联调具备最低闭环：

- [ ] 登录页显示正常
- [ ] 至少一个 OAuth Provider 登录正常
- [ ] 创建新对话成功
- [ ] 至少保存一个可用的模型 Provider API Key
- [ ] 使用已配置 Provider 成功发送一条消息
- [ ] 使用支持视觉的模型成功发送“文本 + 图片”
- [ ] 使用不支持视觉的模型时会被阻止并出现明确提示
- [ ] 刷新页面后历史图片仍正常回显
- [ ] 会话切换正常
- [ ] 会话重命名正常
- [ ] 会话删除正常
- [ ] 退出登录正常

- [ ] Google 登录正常

- [ ] 保存 OpenAI Key
